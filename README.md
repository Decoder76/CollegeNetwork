# CollegeNetwork

Final prototype foundation for a modern student networking application.

## Included in this repository

- `docs/final-prototype.md` — product blueprint aligned to current industry direction and near-term trends.
- `database/schema.sql` — PostgreSQL schema covering identity, profile, social graph, groups, opportunities, messaging, moderation, and auditability.

## Quick start

1. Provision PostgreSQL 14+.
2. Apply `database/schema.sql`.
3. Build API services on top of this schema and expose the v1 endpoints documented in `docs/final-prototype.md`.
